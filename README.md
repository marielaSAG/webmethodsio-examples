# webMethods.io workflow examples
Collection of examples for beginner webMethods.io developers. It consists of common use cases and starting points for developers getting up to speed with webMethods.io. webMethods.io is an integration platform that enables a wide variety of integration tasks to be accomplishes quickly and easily with a graphical/api interface. In many cases, use of webMethods.io allows tasks to be accomplished much faster than traditional efforts involving software engineering, more information can be found here. Each example will contain instructions on how to recreate itself from a blank project.
